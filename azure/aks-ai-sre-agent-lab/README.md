# Azure AKS AI SRE Agent Lab

This lab is part of the Aclyro platform.

## Purpose

Build a real-world Azure-native Python, Kubernetes, DevOps, SRE, AI, and cybersecurity lab using:

- Azure Kubernetes Service
- Azure Container Registry
- Terraform
- Ansible
- Python FastAPI
- GitHub Actions
- Azure DevOps Pipelines
- PostgreSQL
- AI agent workflows
- Observability
- DevSecOps scanning

## Goals

- Provision AKS automatically with Terraform
- Build and containerize a Python FastAPI service
- Deploy the app to AKS
- Pull external data through APIs
- Store memory/cache data in PostgreSQL
- Add AI-assisted SRE incident analysis
- Add cybersecurity scanning and reporting
- Build CI/CD pipelines using Azure-native tooling and GitHub Actions
